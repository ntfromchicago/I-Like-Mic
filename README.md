# I-Like-Mic
Download the notarized Mac app for I Like Mic, which is a tiny menu bar utility app that connects your microphone on Vision Pro to your Mac.

Look at the list of Releases to the right to download a .zip file of the latest version.

This app doesn't do anything unless you get I Like Mic for visionOS:

https://apps.apple.com/us/app/i-like-mic/id6760721210
